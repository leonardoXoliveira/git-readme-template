## Development

### Clone Code

```bash
git clone https://github.com/takebo/git-readme-template.git
```

### Install Dependencies

```bash
cd git-readme-template
npm install
```

### Dev Mode

```bash
npm start
```

### Testing

```bash
npm test
```

### Build Release

```bash
npm run build
```

After building, the application will be found in the project's `build` directory.
